# 🚀 Namaste Tractor – Frontend (React)

## 📌 Overview
Namaste Tractor Frontend is a React-based web application that provides a user interface for managing tractor listings, user authentication, and interacting with backend APIs. It allows users to explore tractors, post listings, and perform basic operations through a clean UI.

---

## 🛠️ Tech Stack
- React  
- JavaScript  
- HTML  
- CSS  
- Axios (API Integration)  

---

## 🎯 Features

### 🔑 Authentication
- User login and registration forms  
- Integration with backend JWT authentication APIs  

---

### 🚜 Tractor Management
- View tractor listings  
- Add new tractor details using forms  
- Display tractor information dynamically  

---

### 🔄 API Integration
- Connected with backend REST APIs  
- Handles data fetching and submission using Axios  

---

### 🧩 UI Components
- Reusable components for forms and layouts  
- Structured pages for better maintainability  

---

### 🔀 Routing
- Implemented client-side routing using React Router  
- Navigation between pages without page reload  

---

## 🧠 Architecture
The project follows a component-based architecture:

Components → Pages → API Services

- Components: Reusable UI elements  
- Pages: Main screens (Login, Dashboard, etc.)  
- Services: API calls to backend
